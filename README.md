# (수정중)

## 시스템블럭
1) 사람이 접근하여 누구인지 확인한다.
2) 인식여부를 확인해서 스피커를 통해 인사한다.
3) 출입자명부를 자동으로 작성해 준다.

## 소프트웨어 개발 WBS
1) 카메라를 이용하여 사람을 구분한다.
2) 스피커로 인식여부를 판단해 인사한다.
3) 로그 데이터를 파일로 만들어 준다.
